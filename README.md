# dotfiles

Deploy packages, e.g. `nvim`, selectively with:

```sh
stow -v <pkg>
```

## Prerequisites

- git
- stow

## Programs

### Terminal

- Shell: Zsh
- Emulator: ghostty
- Prompt: starship
- Package manager: antidote
- Editor: neovim

### Essentials

- bat
- eza
- fd
- fzf
- ripgrep
- zellij
- zoxide

### Utilities

- btop
- direnv
- hexyl
- jless
- jq
- lazygit
- tailspin
